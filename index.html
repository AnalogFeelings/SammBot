About

Samm-Bot is a new Discord bot designed with simple usage in mind, and it's open source! Anyone can contribute to Samm-Bot's development, which by the way, it's in constant development! Although, I can only work on it in my free time, and sadly, school has made this harder for me.

Samm-Bot doesn't need much memory to run, and it's fast. My private official instance runs on a Raspberry Pi 4B, and it uses around 120 megabytes of memory. It should also run on pretty much any OS that supports .NET, including Windows, Linux and macOS.

Keep in mind that I'm the only person working on this project, so sometimes development can be slow. I can't catch all bugs either, so GitHub issues would be really appreciated!

Samm-Bot is licensed under a permissive license, and all the libraries used are open source as well!

Features

Samm-Bot's features are spread apart in "modules". Each module is a set of commands!

⚙

Administration

Allows you to manage the bot through commands!
Including restarting, shutting down, saying messages, and more!

❔

Help

I think this is pretty self-explanatory.
It just lets you get a list of modules and commands!

🎲

Fun

Hug users, pat them, ask the magic 8-ball, and more!

ℹ

Information

Get information from the bot, users, or the current guild!

🧑‍⚖️

Moderation

Allows moderators to kick, ban, mute and
even bulk-delete messages!

👥

Profiles

Per-user profiles!
Currently only supports pronouns. The
bot will use these pronouns in some commands.

🎰

Random

Randomized crazyness!
Get a cat, dog, duck or even fox image! Hell, you can even get a random SCP article.

🏷️

User Tags

Allows users to create tags!
A tag is composed of a name and a reply. When a user gets this tag, Samm-Bot replies with the set reply!

🔧

Utilities

Get user avatars, visualize color codes, or even get your city's weather forecast!

Install

Samm-Bot does not currently have public-available hosting. You will need to download the bot's source code, compile it, and host it yourself. I understand this is not convenient for non-tech-savvy users, but it's all I can do right now.

Head over to the bot's repository to clone it and compile it. You will also need to create an account for the bot in the Discord Developer Portal.

Sorry for the inconvenience! It's just that I don't have money to buy a VPS.
