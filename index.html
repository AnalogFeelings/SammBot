Simplicity and freedom.

Introducing my project, Samm-Bot!
The free, open-source, and easy to use Discord bot of your very own dreams!
Samm-Bot is fully community-maintained and anyone can help with development.

A simple bot.

Samm-Bot is a bot that focuses on being simple to use.
Whether you want to invite it to your server for fun, or for moderation purposes, Samm-Bot will be unintrusive and slick to use for everyone.

No fuss at all.

No advertisements for some sort of premium subscription.
No random pings telling you that you levelled up.
No e-mails to advertise some sort of crappy NFT collection.
Samm-Bot is a very quiet bot that will only interact with you when you interact with it.

Has all the basics.

Samm-Bot is a bot has all the basics you need to moderate your server, and fun commands for casual interactions.
Welcome messages, warning system, muting, logging, and more to come in future updates.

Open source, too.

Samm-Bot is built on .NET technologies, and it is open source.
It always has been, and will always be. Forever. I promise.
The code for both the bot and website are hosted on GitHub, and everyone is able to contribute to their development.
